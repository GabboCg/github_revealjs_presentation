Configurar git en R Studio Cloud

  • Cada proyecto nuevo en R Studio Cloud se debe activar el uso de Git.
  • Tools > Proyects Options > Git/SVN > Version Control System > Git
  • Luego click en yes y nuestro proyecto tendrá disponible las características de Git.

GitHub

Creando nuestro primer repositorio

  • Debemos crear el repositorio desde GitHub.
  • Los camandos necesarios para “linkear” el repositorio con R (R Studio Cloud), los veremos una vez generado el repositorio.

Comandos para usar Git

  • Hasta ahora no hemos realizado ningún commit, por ende, debemos identificarnos.
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
  • Copiamos el código: …or push an existing repository from the command line
git remote add origin https://github.com/TestNCC1801/primer_repositorio.git

Comandos para usar Git

  • Click en Git.
  • Seleccionamos los archivos y escribimos “nuestro primer commit”.
  • Click en commit.
  • Luego en la Terminal escribimos:
git push -u origin master
